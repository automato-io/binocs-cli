## binocs user update

Update current binocs user

### Synopsis


Update any of the following parameters of the current binocs user: 
name, timezone


```
binocs user update [flags]
```

### Options

```
  -h, --help              help for update
  -n, --name string       Your name
  -t, --timezone string   Your timezone
```
